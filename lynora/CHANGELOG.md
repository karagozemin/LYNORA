## [0.1.0] - 2025-01-XX - Wave 5 Initial Release

### Added
- Initial smart contract implementation in Rust
- Market creation functionality on user microchains
- Real-time UP/DOWN betting system
- Cross-microchain market discovery
- GraphQL API with queries and subscriptions
- React frontend with TailwindCSS
- Dynamic Labs wallet integration
- Real-time pool updates via subscriptions
- Oracle integration with CoinGecko
- Proportional reward distribution
- User dashboard for tracking positions
- Market detail pages with live betting
- Autonomous market resolution system
- Comprehensive documentation (DEPLOYMENT.md, DEMO_SCRIPT.md)
- Build and deployment scripts
- README with quick start guide
